# SmartWork Agent Platform Vision

Updated: 20260608-102303

## Tujuan Besar
SmartWork Agent bukan hanya aplikasi SIAGA. SIAGA adalah agent pertama/prototype.

SmartWork Agent akan menjadi platform multi-agent untuk membantu pekerjaan admin, absensi, operator, guru, dan sekolah.

## Prinsip Produk
- Banyak agent, bukan satu script.
- Setiap agent punya workflow sendiri.
- Agent harus modular.
- User harus bisa memilih agent dan tugas.
- Save, delete, submit, kirim data wajib konfirmasi user.
- Semua aksi penting harus punya screenshot dan report.
- Data sensitif tidak boleh masuk GitHub.
- Untuk public app, credential harus aman dan terenkripsi.
- App harus bisa dipakai pribadi dan nanti bisa dipakai banyak orang.

## Contoh Agent
- SIAGA Agent
- Absensi Agent
- Operator Sekolah Agent
- File Admin Agent
- Report Agent
- Screenshot Agent
- Guard Agent
- SmartBrowser Agent

## Arah UI
SmartWork Control Panel nanti menjadi dashboard/marketplace agent:
1. Pilih agent.
2. Pilih tugas.
3. Lihat rencana kerja.
4. Jalankan agent.
5. Lihat progress.
6. Approve save/delete/submit jika perlu.
7. Lihat screenshot dan report.

## Catatan
Jangan hardcode semua sebagai SIAGA. SIAGA harus dipindahkan menjadi satu module agent.
