# SIAGA Absensi Jam Rules

Updated: 20260608-093826

## Jam Masuk
- Semua hari kerja: 06:50 sampai 07:00.
- Tidak boleh lewat dari 07:00.
- Menit dibuat acak agar tidak monoton.

## Jam Pulang
- Senin-Kamis: 14:15 sampai 14:30.
- Jumat: 11:30 sampai 11:35.
- Sabtu: 15:15 sampai 15:30.

## Aturan Agent
- Diagnose DOM input jam dulu sebelum fill otomatis.
- Jangan ubah zoom/viewport.
- Jangan simpan sebelum user mengizinkan.
- Gunakan random menit yang tetap dalam batas aman.
