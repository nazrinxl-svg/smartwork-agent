Alur SmartWork

1. Isi form
User isi email, WhatsApp, bulan/tahun, dan akun guru.
2. Agent bekerja
SmartWork menjalankan proses SIAGA sesuai request. Save/submit/kirim tetap pakai guard.
3. Bukti hasil
Setelah selesai, SmartWork siapkan PDF presensi dan laporan bukti kerja.
4. Delivery
PDF + laporan bukti disiapkan untuk email dan WhatsApp.
Command setelah request siap: npm run intake:validate npm run batch:run npm run siaga:proof-delivery
